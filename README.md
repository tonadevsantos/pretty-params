# Pretty Params

Manage URL query strings with an automatically generated form.

## Development

## Node version

> v20.13.1

Install dependencies:

```sh
npm install
```

### Run in Chrome mode:

```sh
npm run dev -- --mode=chrome
```

### Run in Firefox mode:

```sh
npm run dev -- --mode=firefox
```

## Build

### Build for Chrome

```sh
npm run build:chrome
```

### Build for Firefox

```sh
npm run build:firefox
```
